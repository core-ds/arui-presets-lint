## Разработка

Быстрый старт:

```bash
yarn install --immutable
yarn build
yarn test
yarn lint
```

Подробнее тут: [CONTRIBUTING.md](./CONTRIBUTING.md)

Проект является монорепозиторием, в котором находятся следующие пакеты:

- [arui-presets-lint](./packages/arui-presets-lint/) - Конфигурация линтеров
- [stylelint-core-vars](./packages/stylelint-core-vars/) - Плагин stylelint для проверки дизайн-токенов
